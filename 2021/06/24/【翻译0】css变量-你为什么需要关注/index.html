【翻译】CSS变量:你为什么需要关注?

原文 CSS Variables: Why Should You Care?

  CSS 变量，更准确地称为 CSS 自定义属性，将登陆 Chrome 49。它们可用于减少 CSS 中的重复，以及强大的运行时效果，如主题切换和潜在的扩展/填充未来的 CSS 功能。

杂乱的 CSS

  在设计应用程序时，通常的做法是留出一组品牌颜色，这些颜色将被重复使用以保持应用程序的外观一致。不幸的是，在 CSS 中一遍又一遍地重复这些颜色值不仅是一件苦差事，而且容易出错。如果在某个时候需要更改其中一种颜色，您当然可以进行全局“查找并替换”所有值，但在足够大的项目中，这很容易出错。

  最近，许多开发人员转向了像 SASS 或 LESS 这样的 CSS 预处理器，它们通过使用预处理器变量来解决这个问题。虽然这些工具极大地提高了开发人员的生产力，但它们使用的变量存在一个主要缺点，即它们是静态的，不能在运行时更改。添加在运行时更改变量的功能不仅为动态应用程序主题等打开了大门，而且对响应式设计和polyfill未来CSS功能的潜力也有重大影响。随着 Chrome 49 的发布，这些功能现在以 CSS 自定义属性的形式提供。
